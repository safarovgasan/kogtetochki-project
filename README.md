# Kogtetochki Project

Проект про когтеточки: ассортимент, визуальные материалы, описания товаров и идеи для продвижения.

## Цель

Создать аккуратную базу для проекта, связанного с когтеточками:

- каталог моделей и материалов;
- тексты для карточек товаров;
- идеи для инфографики и маркетплейсов;
- требования к фото, размерам и упаковке;
- задачи для сайта, магазина или презентаций.

## Структура

- `docs/brief.md` - описание проекта, аудитории и первых задач.
- `docs/tasks.md` - список задач для развития проекта.
- `assets/` - изображения, референсы и готовые материалы.
- `AGENTS.md` - инструкции для Codex при работе с этим репозиторием.

## Быстрый старт

1. Добавить описание конкретных когтеточек в `docs/brief.md`.
2. Положить фото и референсы в `assets/`.
3. Записать задачи в `docs/tasks.md`.
4. Подключить репозиторий к GitHub, если нужен Codex Cloud.
