# Бриф проекта

## О чем проект

Проект посвящен когтеточкам для кошек: разработке ассортимента, оформлению карточек товаров, инфографике, описаниям, упаковке и продвижению.

## Возможные направления

- Настенные когтеточки.
- Напольные когтеточки.
- Когтеточки-столбики.
- Когтеточки-домики.
- Картонные когтеточки.
- Джутовые и сизалевые модели.
- Модели для маркетплейсов.

## Аудитория

Люди, у которых есть кошки и которые хотят защитить мебель, дать питомцу удобное место для стачивания когтей и выбрать вещь, которая нормально выглядит дома.

## Что важно подчеркнуть

- Материалы.
- Размеры.
- Устойчивость.
- Безопасность.
- Простота сборки.
- Для каких кошек подходит.
- Как вписывается в интерьер.
- Чем лучше обычной альтернативы.

## Первые решения

- Название проекта: уточнить.
- Формат проекта: каталог, сайт, презентация, маркетплейс или бренд.
- Основной канал: уточнить.
- Нужны ли фото, инфографика, карточки товаров или тексты.
